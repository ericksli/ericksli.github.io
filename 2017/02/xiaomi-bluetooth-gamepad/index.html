小米藍牙手柄

February 4, 2017

之前提及過只需要在 Windows 安裝 mapper 就能將小米藍牙手柄模擬成 Xbox 360 手掣，試過之後確實可以用到。在 The Crew 可以用手掣操作，在刹掣時手掣還會震動，比起在 Android 用功能還多。

除了 desktop game 之外，原來 HTML5 有 Gamepad API。我特別在 HTML5 Gamepad Tester 試了一次（未有用 Xbox mapper）。
